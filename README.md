# 3DSUSBController-Server
3DS Homebrew to turn your 3DS into a USB game controller (Java server code)  

Requires ```jcrm1/3DSUSBController-Homebrew``` and ```jcrm1/3DSUSBController-USB-Device```  

Running:
1. Connect USB device  
2. Start server  
3. Start homebrew  

Building:  
```./gradlew shadowJar```
